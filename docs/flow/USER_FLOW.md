## 상태 전이

```
          ┌──────────────┐
          │  onboarding  │
          └──────┬───────┘
                 │ (자동 전환)
          ┌──────▼───────┐
          │    input     │
          └──────┬───────┘
                 │ (고민 제출)
          ┌──────▼───────┐
          │   loading    │ ←── API 호출
          └──────┬───────┘
                 │ (매칭 완료)
          ┌──────▼───────┐
          │    select    │
          └──────┬───────┘
                 │ (캐릭터 선택)
          ┌──────▼───────┐
    ┌─────│    detail    │
    │     └──────┬───────┘
    │            │ (대화 시작)
    │     ┌──────▼───────┐
    └────▶│     chat     │
   (뒤로) └──────────────┘
```

## 에러 처리

| 상황 | 처리 |
|------|------|
| API 타임아웃 | 로딩 화면에서 재시도 안내 |
| 네트워크 오류 | 토스트 메시지 표시 |
| 빈 입력 제출 | 버튼 비활성화 |
