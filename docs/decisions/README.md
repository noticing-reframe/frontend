# Architecture Decision Records (ADR)

AI와의 협업을 통해 내린 기술적 결정을 기록합니다.

## ADR 형식
```markdown
# ADR-[번호]: [제목]

- **상태**: 제안됨 / 승인됨 / 폐기됨
- **날짜**: YYYY-MM-DD
- **의사결정자**: [이름/팀]

## 맥락
[왜 이 결정이 필요했는지]

## 고려한 옵션
1. **옵션 A**: [설명]
   - 장점: ...
   - 단점: ...

2. **옵션 B**: [설명]
   - 장점: ...
   - 단점: ...

## 결정
[선택한 옵션과 이유]

## AI 협업 내용
[Claude와 어떤 논의를 했는지, 어떤 제안을 받았는지]

## 결과
[이 결정으로 인한 영향]
```
