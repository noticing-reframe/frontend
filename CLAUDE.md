## AI 협업 문서화 구조

```
frontend/
├── CLAUDE.md          # 이 파일 - Claude Code 컨텍스트
└── docs/
    ├── ai-sessions/   # AI와의 세션별 대화 기록
    │   └── YYYY-MM-DD-주제.md
    ├── prompts/       # 재사용 가능한 프롬프트 모음
    │   └── prompt-name.md
    └── decisions/     # AI 협업으로 내린 기술 결정
        └── ADR-001-제목.md
```
